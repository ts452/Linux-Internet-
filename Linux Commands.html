Common Commands

The following are a few LINUX Commands

  1. pwd - print working directory
  2. mkdir - make directory
  3. cd-change directory
  4. ls-list files
  5. cp= copy files/folder(s)
  6. mv - move/rename files/folders
  7. rm - delete files/folders
  8. cat = catenate (concatenate) - reads one or more files and prints them.
  9. chmod- change file`s access permissions(r,w,o)
  10. chown- change owner of file
